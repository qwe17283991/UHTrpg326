經驗值與支線

貨幣介紹與獲取：

===========================================================================================================

在無限恐怖TRPG中，有三種主要貨幣：

支線：支線分為5種，分別為DCBAS。多數情況下，一個標價為C的能力幾乎一定會比一個標價為D的能力要強大，而一個標價為B的能力也會相應比C能力更強。擁有一整套S級能力並且用他們打出一套華麗的連鎖是每一個玩家的夢想。支線可以自由地拆分或組裝。3個D可以拼成一個C，然後3個C可以拼成一個B，以此類推。同樣的，反向的拆解也是可以進行的，一個C可以被拆開來購買3個不同的D支線資源。

獎勵點數：又被稱作分數，是無限流TRPG裡最為常見的貨幣。一般情況下，你所能購買的資源都需要支付一定的支線和獎勵點數。比如一杆高能粒子脈衝步槍的標價為D+500，這就意味著你需要支付一個D支線和500獎勵點數來換取這一昂貴的未來科技武器。

經驗：又稱XP。XP對應表現在冒險世界中透過歷練提升自己的實力。顯而易見地，一個人獲得的XP不可能轉讓給另外一個人。XP也不可能用來從無到有地生成某項能力，而是必須要在有一定基礎的情況下進行“鑽研”或“訓練”來提升。更多的內容可以在“經驗值系統”與“自創能力規則”這兩頁進行查詢。

輪迴之境不僅可以讓參加者們在九死一生的冒險後好好休息，也提供了讓他們變強的方法。在無限流TRPG中，你可以兌換幾乎無限種類的不同能力。

消耗支線與獎勵點數除非特殊情況，則必須在輪迴世界才可以，兌換資源是一個自由動作。

消耗經驗則可以在輪迴世界和冒險世界中都可以直接消耗，若選擇在冒險世界中，則沒有特殊情況（如：光戰，人皇）則需要消耗原本*2的經驗，消耗經驗習得新的能力無需動作。

===========================================================================================================

支線與分數獎勵系統

在無限流TRPG遊戲規則之中，當角色擊敗了怪物或者完成了一項具有難度（一般是指有一定的生命危險）的挑戰之後，我們將發放給玩家一些獎勵，這些獎勵是以支線和分數的形式出現的。

支線和分數可以用於購買輪迴之境的各種強化，包括血統，改造，瞳術，稱號，流派，魔導書等等。

===========================================================================================================

支線與分數的獲得：

1. 擊敗眼前的怪物，根據怪物可以對角色造成的威脅程度來定義該怪物的獎勵。獎勵可以是浮動的，因為同一只怪物在面對不同時期甚至不同角色的時候會造成的威脅不同。

2. 解決眼前的難題，根據難題的困難程度以及必要程度和對角色的威脅定義該獎勵的多少。如在哈利波特中摧毀死亡聖器等。

3. 安全度過一個冒險世界，完成主線任務返回輪迴之境，這是一個固定獎勵。

4.  完成主神發放的支線任務，這可以是任何任務，取決於ST的安排。

===========================================================================================================

如何評估當前的難度：

1. 毫無威脅：怪物或者難題幾乎無法對角色造成任何傷害和影響。獎勵一般為1-10分左右，一般出現的是龐大數量的小怪。

2. 非常簡單：怪物或者難題不會給他們帶來嚴重的障礙，最多損失一點資源（如生命，能量池等），獎勵一般為50-500分左右，一般出現的是小怪中的精英單位。

3. 勢均力敵：怪物或者難題與角色們水平相當，甚至會發生一些驚險時刻，但是勝利的天平依然倒向角色們，獎勵一般為D-C左右，一般出現的是小BOSS了。

4. 頗具挑戰：怪物或者難題明顯比角色們要強大，角色們要承擔一定的風險才有可能戰勝它，這可能會導致隊伍中角色的嚴重受傷或者死亡。獎勵一般為C-B左右，一般就是劇情中的大BOSS了。

5. 致命威脅：怪物或者難題的水平遠超角色們，角色們與其對敵需要良好的策略與戰術，否則會有嚴重的傷亡，甚至有團滅（全員陣亡）的風險。獎勵一般為B級以上，一般是劇情中隱藏起來極其可怕的BOSS。

===========================================================================================================

獎勵的發放建議：

支線與獎勵點數可以任意進行拆分，如一個C級支線可以拆分成3個D級支線，1000分可以用來購買2分500分的物資，可以購買10分100分的物資。

當然，ST有權禁止玩家進行拆分，規定玩家只能按照其所獲得的支線等級進行購買（分數依然可以拆分），但是我們並不建議這麼做，畢竟自由度是本規則的魅力所在。

我們並不支援將獎勵全部給與一個玩家，建議玩家可以平均分配，當然如果有特殊需求可以優先安排，這是玩家的事情並不過多幹擾，但是一般情況下，一個角色如果獲得的獎勵遠超其他玩家，那麼會影響其他玩家的遊戲體驗。

這由ST靈活掌握，是否強行平均分配又或者是禁止玩家之間贈送或者交易支線或者分數。

===========================================================================================================

固定的獎勵發放：

當玩家成功返回輪迴之境時，其將獲得n/2個D級支線獎勵。n為冒險世界的次數，向上取整。然後每有一個D，獲得1000分數。

例：

第一場冒險世界：D級支線獎勵+1000分

第二場冒險世界：D級支線獎勵+1000分

第三場冒險世界：DD級支線獎勵+2000分

第四場冒險世界：DD級支線獎勵+2000分

第五場冒險世界：C級支線獎勵+3000分

第六場冒險世界：C級支線獎勵+3000分

第七場冒險世界：CD級支線獎勵+4000分

第八場冒險世界：CD級支線獎勵+4000分

第九場冒險世界：CDD級支線獎勵+5000分

以此類推

===========================================================================================================

支線與分數的作用：

用於購買輪迴之境的兌換，如沒有特殊說明，支線和分數只能在輪迴之境使用：

1. 改變自身的屬性。

2. 購買物品。

3. 購買血統，改造，瞳術，典籍，修煉體系五大模板類能力。

4. 購買稱號，流派等職業性質能力。

5. 購買技能，法術，招式等應用性質能力。

===========================================================================================================

經驗值系統

經驗值（XP）是在經歷過各種事情之後人物獲得的某種提升，擁有常人所沒有的非凡經歷的人物自然也擁有比一般人更多的歷練、經驗和知識。經驗值可以提升人物的各種技能，獲得更多的技巧（專長），刻苦的鍛鍊還可以增強人物身體與心靈的強度。此外，輪迴之境所給予的某些強化也需要額外的經驗去理解、練習才能真正掌握。

===========================================================================================================

經驗值的獲得：

在無限恐怖TRPG中，每個玩家所獲得的經驗值根據其表現的優劣而定。一般來說，ST應該在每次跑團結束後，根據各人該次的表現記下其獲得的經驗；在每次冒險結束後，則再根據各人在該冒險世界中整體的表現給予其經驗。在大多數情況下，ST可以在一次冒險世界結束的時候統一將這些經驗發放；而若是玩家的人物在冒險世界（或冒險之外的某個章節）當中獲得了較長的時間以休息、鍛鍊，ST也可以將其已獲得的經驗先行發放，以使其增加自己所需要鍛鍊的技能。

一般來說，經驗透過以下方式發放：

將經驗分為5個部分，根據PC的表現將5個部分相加，總計為PC經驗。

第一部分：基礎值，基礎值為當前冒險世界的場次來決定，其在返回輪迴之境時發放，所有角色獲得2+2n點經驗值，n為冒險世界的次數。

第一場冒險世界：4點XP，第二場冒險世界：6點XP，以此類推。

第二部分：獎勵經驗，獎勵經驗是計算本場所有角色獲得的所有分數和支線的總計（計算固定獎勵），角色們每獲得了一個D，則獲得3點經驗，每有500分則獲得2點經驗，然後根據參團PC人數除以這個數，向上取整，每個PC獲得對應的獎勵經驗，其在返回輪迴之境時發放。

（如，角色們總計獲得BB+16000，則兌換成分數是54+48=102點，然後一共4個PC，則每個人獲得25點XP，向下取整。）

第三部分：特殊表現，該經驗的獎勵內容如下，其在返回輪迴之境時發放：

1. 參團率：該PC完整的參與了一場跑團，期間沒有掛機託管等行為，並且進行了基礎的行動，給與2點經驗。（該經驗在當次跑團結束後立即發放）

2. 參戰率：該PC完整的參與了一場戰鬥，並且這場戰鬥的評估水平在“勢均力敵”以上，同時該PC進行了與戰鬥有關的對應行動，如承擔傷害，造成傷害，提供BUFF等，則其獲得1點經驗。（一般而言，一場團的戰鬥次數不建議超過5次）。

3. 表現獎勵，表現獎勵是ST根據玩家在團內的表現發放的額外獎勵，表現包含：戰力擔當、RP突出、劇情表現等。為了有表彰的含義，所以一般根據最後存活下來的PC總人數-1來發放。比如4個PC，表現最好的給予3點，一般的給予2點，較差的給予1點，最差的不給。比如6個PC，則第一就是5點經驗。

4. 劇情懲罰：每有一個PC陣亡，則活著的所有PC扣除1點經驗。一個設計好的支線失去或者重要NPC死亡，所有PC扣除1點經驗。

特殊：如果該團的經驗和支線獎勵發放較多，則第三部分的獎勵與懲罰可以乘以一個係數，該係數為已經歷過的冒險世界的場地，如第一場為1，第二層為2，這樣經驗獎勵和懲罰也可以有所增長。

===========================================================================================================

經驗值的作用：

1. 購買自身的屬性：。

2. 購買自身的技能等級。

3. 購買專業。

4. 購買專長。

5. 購買稱號，流派等職業性質能力。

6. 購買技能，法術，招式等應用性質能力。

特殊：需要注意的是，經驗與支線和分數不同，可以不在輪迴之境使用，而是在冒險世界中使用，但是若在冒險世界中使用，則需要花費原本經驗值的二倍。
